# Project Management System - Frontend

A React.js application for managing Customers, Projects, and Employees.

## 🚀 Technologies Used

- React.js
- React Router
- Axios
- Bootstrap
- React Icons
- React Toastify

## 📌 Features

- Dashboard
- Customer Management
- Project Management
- Employee Management
- CRUD Operations
- Search Functionality
- Responsive UI
- REST API Integration

## Screens

- Dashboard
- Customers
- Projects
- Employees

## Backend

Spring Boot REST API

Runs on

```
http://localhost:8080
```

Frontend

```
http://localhost:5173
```

## Installation

Clone repository

```
git clone https://github.com/PrathameshDekhale/project-management-system-frontend.git
```

Install packages

```
npm install
```

Run project

```
npm run dev
```
